# chatbot-gordo-de-closet

## Uso

1. Envía un mensaje con el formato:

   ```
   Hola, deseo reservar en NOMBRE_DEL_RESTAURANTE
   ```

   El bot buscará ese restaurante en la base de datos y, si lo encuentra, responderá con los detalles e imagen del lugar.

2. Si el restaurante indicado no existe, recibirás un aviso y se mostrará un listado de restaurantes disponibles para que elijas.

3. También puedes escribir simplemente `Hola` para obtener el listado de restaurantes y seleccionar uno desde la lista interactiva.
